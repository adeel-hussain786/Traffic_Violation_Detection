# 🚦 Traffic Violation Detection – **TrafficGuard**

**TrafficGuard** is a **React Native–based mobile application** developed for **traffic violation detection and monitoring**.
It serves as a scalable **frontend solution** that can be integrated with backend services and **AI/ML models** to support intelligent traffic management.

---

## 📌 Key Features

* 📱 Cross-platform mobile application (**Android & iOS**)
* ⚡ Built using **React Native CLI**
* 🧠 Designed to support **AI-based traffic violation detection**
* 🧩 Clean, modular, and scalable architecture
* 🔒 Ready for backend, API, and database integration

---

## 🛠 Technology Stack

* **Framework:** React Native (CLI)
* **Language:** JavaScript / TypeScript
* **Package Manager:** npm / yarn
* **Platforms:** Android & iOS

## 🚀 Getting Started

### ✅ Prerequisites

Make sure the **React Native development environment** is set up correctly.

🔗 [https://reactnative.dev/docs/environment-setup](https://reactnative.dev/docs/environment-setup)

Follow the steps **up to “Creating a new application”**.

---

## ▶️ How to Run the Application

### 1️⃣ Install Dependencies

```bash
npm install
# or
yarn install
```

---

### 2️⃣ Start Metro Bundler

```bash
npx react-native start
```

Keep this terminal running.

---

### 3️⃣ Run on Device / Emulator

#### 📱 Android

```bash
npx react-native run-android
```

#### 🍎 iOS (macOS only)

```bash
npx react-native run-ios
```

---

## ✏️ Modifying the App

1. Open `App.js`
2. Make changes and save

### 🔄 Reload the App

* **Android:** Press `R` twice or `Ctrl + M`
* **iOS:** Press `Cmd ⌘ + R`
